# Code of Conduct

This project follows the [Contributor Covenant
v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

- Be kind. Assume good intent. Keep discussions technical.
- Harassment, discrimination, personal attacks, and gatekeeping are
  not welcome.
- Maintainers may remove comments, lock issues, or ban contributors
  who violate this code.

## Reporting

Report violations via GitHub's
[abuse reporting](https://docs.github.com/en/communities/maintaining-your-safety-on-github/reporting-abuse-or-spam)
flow, or by email to `conduct@<your-domain>` (replace with the
maintainer email when you fork).

Reports are confidential. We aim to respond within 7 days.

## Scope

This code applies to all project spaces (issues, PRs, discussions,
chat, events) and to public conduct that materially affects the
project's community.
